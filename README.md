 Password Generator

A simple and responsive Password Generator built using **HTML, CSS, and JavaScript**.  
Allows users to generate strong passwords with customizable length and character options.



Live Demo :- https://supriyode2002.github.io/Javascript-Project--Password-generator/


 Features
- Generate secure random passwords  
- Customize length and character types  
- Copy password to clipboard  
- Clean and user-friendly UI  
- Fully client-side and fast


 Tech Stack
- HTML  
- CSS  
- JavaScript
